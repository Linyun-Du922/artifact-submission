#  Artifact Package

## Overview

The artifact package includes the Reddit multi-stage annotation data, the temporal-structure dataset, and qualitative study materials. 

## Contents

### Reddit multi-stage annotation data.csv

This file contains the anonymized post-level annotation data used to analyze recurring interactional stages in sextortion-related Reddit narratives.

The dataset includes derived annotation labels for the interactional stages discussed in the paper. Raw Reddit post titles, post bodies, usernames, URLs, and other direct identifiers have been removed.

### temporal structure dataset.csv

This file contains the anonymized temporal-structure dataset used to analyze the ordering and progression of interactional stages.

The dataset includes derived temporal stage sequences and related annotation variables. Raw Reddit post titles, post bodies, usernames, URLs, and other direct identifiers have been removed.

### semi-structured interview protocol.pdf

This file contains the semi-structured interview protocol used for the qualitative interview study. It includes the interview questions and prompts used to elicit participants' experiences, responses, and reflections.

### Informed Consent Statement.pdf

This file contains an anonymized consent/information sheet for interview participants. Author names, institutional information, contact information, and IRB-identifying details have been anonymized for double-blind review.

## Privacy and Anonymization

To protect Reddit users and interview participants, this artifact does not include raw Reddit narratives, usernames, profile links, URLs, direct contact information, or other identifying information.

## Notes for Reviewers

This artifact is provided for double-blind review. All uploaded files have been reviewed to remove information that could identify the authors, institution, research team, or participants.

The materials are intended to support evaluation of:

- the interactional-stage annotation scheme;
- the temporal-structure analysis of Reddit narratives;
- the qualitative interview protocol;
- the ethical and consent procedures used in the interview study.

## Limitations

For ethical and privacy reasons, the artifact does not release raw Reddit post text or full interview transcripts. These materials may contain sensitive victim disclosures and potentially identifying information.
